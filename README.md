# ajax-form

webcomponent to submit from data through ajax request


## Usage

```html

    <h2>ajax-form element demo</h2>

    <ajax-form>
        <form action="/post">
            <input type="hidden" id="name" value="myname" >
            <input type="text" id="value" />
            <input type="submit" text="Submit" />
        </form>    
    </ajax-form>
```
